## Typography

### Font Families
- **Primary Font**: `TP Sans`
- **Secondary Scale**: `Major Second (1.125)` with `1.2 line height` and `0.3px kerning`

### Responsive Typography Breakpoints
- **Mobile**: `767px` below
- **Tablet (md)**: `768px+`
- **Desktop (lg)**: `1024px+`
- **Large (xl)**: `1280px+`
- **XLarge (2xl)**: `1536px+`

### Font Sizes & Line Heights

#### Display Styles
- **Display XL**:
  - Mobile: `28px / 1.75rem`
  - Tablet: `32px / 2rem`
  - Desktop: `36px / 2.25rem`
  - Large: `40px / 2.5rem`
  - XLarge: `44px / 2.75rem`

- **Display L**:
